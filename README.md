# 🇮🇳 Tribute Page – Indian Armed Forces

A responsive tribute website dedicated to the **Indian Armed Forces**, honoring the courage, sacrifice, and dedication of the brave soldiers who protect our nation.

## ✨ Features

- 🇮🇳 Beautiful Hero Section
- 🪖 Indian Army, Indian Navy & Indian Air Force
- 📜 Military Operations
- 🕒 Historical Timeline
- 🖼️ Interactive Gallery
- 📱 Fully Responsive Design
- 🎨 Modern & User-Friendly Interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
